pela FATEC Zona Leste.

O propósito deste programa é guardar
dados de armas em obras de ficcção:
- Quem usa as armas, personagens é 
claro;
- Em que obra estes personagens estão
presentes;
- Quem fabrica esta arma;
- Qual o calibre da arma.

Faça bom uso, Deus vos abençoe!
(independente de crer N'Ele ou não)

Pré-Requisitos:
- 1 PC (dã) com 1GB de RAM e Processador
com 1GHz;,
- Banco de Dados MySQL instalado nesta
máquina;
- Java instalado (Java 8 de preferência).

Para instalar:
1 - Descompacte o arquivo 
Projeto ArmedCharacters.zip;
2 - Abra o ArmedCharacters.sql no
bloco de notas;
3 - Copie o texto inteiro;
4 - Abra o MySQL <Sua versão> Command
Line Client (Iniciar > Todos os Programas > 
MySQL > MySQL Server <Sua versão> >
MySQL <Sua versão> Command Line Client);
5 - Digite sua senha de usuário root;
6 - Cole o script copiado do
ArmedCharacters.sql (botão direito do mouse
 > colar) Nota: vai por mim o DOS só cola assim.
7 - Necessitando, aperte "Enter";
8 - Execute o ArmedCharacters.jar;
9 - Digite sua senha de root no MySQL;
10 - Está pronto para uso, aproveita! ;)

Nota: O programa Java.sh (criado por Caio de
Oliveira Reis) é para contagem de linha, para
este funcionar seu projeto tem que estar dentro
da biblioteca de Documentos do Windows. E
você pode ver o projeto deste sistema e o
modelo de banco de dados.
